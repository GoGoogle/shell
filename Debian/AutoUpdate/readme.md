  ### 有关SHELL的自动检测更新模块
  
  - #### 先前一直在想如何才能自动更新一些想法
  1. 在每三分钟执行一次的前提条件下（crontab);
  2. 脚本内要有个内置的时间范围判断条件;
  3. 如果满足条件2的时候，再拉取远程shell，以免频繁拉取远程文件;
  4. 从远程shell文件内查找新的版本;
  5. 对比当前shell的版本，不一样就开始下载保存;
  6. 下一个3分钟执行脚本的时候就是最新的版本了.
  - #### 今天我才发现这个思路有些不对
  1. 例如步骤2和3中的提取远程shell,都已经拉取了全部内容，为何不直接下载保存?
  2. 所以这个自动检测更新的思路还是留下来给别的情况下使用吧.
